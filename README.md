# Korea Trip Picker 🇰🇷

An interactive activity picker for our Nov 2026 South Korea trip
(Seoul · Gyeongju · Busan · Jeju). Tap the box to collect an option,
tap the card for details, map, cost and a tickets link. Hit **Export**
to copy the group's picks.

## Hosting on GitHub Pages
1. Put `index.html` in the repo root (already here).
2. Repo **Settings → Pages → Build and deployment**.
3. Source: **Deploy from a branch**, Branch: **main** / **/(root)**, Save.
4. Wait ~1 min — your site appears at
   `https://<your-username>.github.io/<repo-name>/`

That's it. GitHub Pages serves over HTTPS, so the "Export / copy picks"
button works for everyone.
